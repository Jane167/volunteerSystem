# 基于 DRF + React的疫情防控社区志愿者管理系统

## 后端project_drf

```python shell
# install requirements
pip install -r requirements.txt

# database migrations
python manage.py makemigrations
python manage.py migrate

# runserver at 127.0.0.1:80
python manage.py runserver 80
```

## 前端project_react

```shell
# 安装依赖
yarn
# 启动服务
yarn start
```
